Category Creation Tips
Good Examples:
  • ✓ Mathematics - Calculus, Algebra, Geometry
  • ✓ Physics - Mechanics, Thermodynamics
  • ✓ Programming - Python, JavaScript, Web Dev
Avoid:
  • ✗ Stuff - Too vague
  • ✗ My Files - Too personal
  • ✗ 12345 - Not descriptive
Category Guidelines:
  • Use clear, specific names that reflect the subject matter
  • Organize by academic subject (Mathematics, Physics, Literature)
  • Or organize by content type (Lecture Notes, Exam Papers, Tutorials)
  • Add descriptions to help others understand what belongs here
  • Categories are visible to all users - keep them general and useful
  • You can edit category details later, or delete them if you're an admin
